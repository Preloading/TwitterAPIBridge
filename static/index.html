A Twitter API Translator

Specifically Bluesky -> Twitter API V1


What is this?

This is a server that allows you to access Bluesky using the Twitter API format. This allows old twitter clients to be useful again, and gives a way to browse Bluesky using a native app, without needing to write a custom one.
