What is this?

This is a server that allows you to access Bluesky using the Twitter API format. This allows old twitter clients to be useful again, and gives a way to browse Bluesky using a native app on old devices, without needing to write a custom one.

Usage Instructions

Official iOS app

  1. Install Twitter. You can find versions of twitter here. Recommended versions are 4.1.3, and 5.0.3
  2. Create a Bluesky app password. You can find this here. Enabling DMs is currently optional, though will be used for future versions.

    or do this thru Bluesky > Settings > Privacy and security > App passwords

  3. Open the Twitter app on your device.
  4. Press the Login button (or signin depending on version)
  5. Press the cog/settings menu
  6. Put {{.PrefixedURL}} into the URL spots. Other public instances can be found on the twitter bridge github.
  7. Go back to the login prompt.
  8. Enter your bluesky handle for the username, and your Bluesky app password as the password. Usage of normal passwords is not recommended, and will be forbidden in the future
  9. Press the sign in button
  10. If everything worked, you should be in! Enjoy!

iOS Twitter Integration + image uploads in app

Thanks to bag.xml & Requis, a Cydia tweak now exists to redirect the iOS integration, and to enable for image uploads.

  1. Have a jailbroken device. For intergation, it should be running iOS 5-7. The min iOS version for image uploads isn't known right now.
  2. Add either http://cydia.bag-xml.com or http://cydia.skyglow.es to Cydia's sources tab.
  3. Install the package Bluetweety from the repo.
  4. Go to Settings > Bluetweety. (if it does not show up, check that it actually installed)
  5. Input {{.UnPrefixedURL}} into the URL. Do NOT include https:// at the start, and a / at the end!!!!
  6. Reboot your phone. Yes this is a required step.
  7. Go to Settings > Twitter.
  8. Type your bluesky handle (example: preloading.bsky.social) and your app password (can be the same from the app) in the respective areas.
  9. Click Login
  10. Scroll down, and disable the ability for the Twitter app to view Twitter accounts.
  11. Hopefully success!

FAQ.

What is this compatible with?

A list of tested & compatible twitter clients can be found on my github, or by clicking here. If it's not there try it and see! (and maybe make a PR of the outcome). In general, this bridge supports around 2010-2012.

Bluesky is cool and all, but I want it to be actually Xitter!

There are various reasons why having this be with twitter data is infeasible, and would not work. This includes:

  1. Running it (and making it) would cost $5000/month, or potentially even more!
  2. Twitter has become a hellscape.
  3. Sign in would likely be extremely hard/impossible.
  4. Twitter has been extremely hostile to third party clients

I found the mobile app API keys! Can't you use those!??!?!?

No.

  1. We would be violating the TOS of Xitter using these keys, we would get IP banned, and potentially the users of the service banned.
  2. It would have to be self-host only. Any pre-hosted version would get IP banned extremely quickly
  3. These keys have restricted access, and only allow internal API points, which have little to no documentation.

Can you do {INSERT OTHER SOCIAL MEDIA APP}

Probably not. I have some plans to potentially do mastodon in the far future, but anything other than that, no.
